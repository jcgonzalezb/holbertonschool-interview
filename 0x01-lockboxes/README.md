# Python - Lockboxes

## Description of the files inside this folder:

0. Write a method that determines if all the boxes can be opened. There are n number of locked boxes. Each box is numbered sequentially from 0 to n - 1 and each box may contain keys to the other boxes. Boxes is a list of lists. A key with the same number as a box opens that box. All keys will be positive integers. There can be keys that do not have boxes. The first box boxes[0] is unlocked. Return True if all boxes can be opened, else return False.
